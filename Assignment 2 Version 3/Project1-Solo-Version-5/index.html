Shaw Website Application

During my co-op at Dr.Logicks Internet Cafe there were days where I had nothing to do but sit back and watch the computers download their updates. So I decided to spend my free time building an application to help make the tiresome task of manually collecting data easier. This website allowed my manager to input information about the client into a local database where she could then extract the formatted data into an Excel sheet and email it to the partner company to receive the rebate. The program was created using ASP.NET and an sql database, I chose this because after I left the co-op position my boss could edit and manage the site with her background in web design.
